# cursor-optimization-framework
Фреймворк для детерминированного управления контекстом в Cursor. Переносит состояние сессии в файловую структуру, изолирует задачи через микро-контексты и синхронизирует координаты функций через git-снапшоты. Предназначен для снижения нецелевого расхода токенов в Agent Mode.
